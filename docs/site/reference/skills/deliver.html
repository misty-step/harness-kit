deliver

Inner-loop composer for one backlog item to merge-ready code. Composes /shape, /implement, /code-review, /ci, /refactor, /qa, and /demo evidence; stops before push, merge, or deploy. Emits receipt.json, evidence packet, delivery brief, and learning packet. Use for "deliver this", "make it merge-ready", shaped-ticket builds, and `--polish-only <branch|PR>` for existing branch/PR cleanup. Trigger: /deliver.
