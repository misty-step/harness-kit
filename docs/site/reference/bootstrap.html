Bootstrap behavior

bootstrap.sh installs the system-wide harness links and provider roster. Consumer repos can then use Spellbook without vendoring source-repo skill bridges.

System install

Installs first-party skills and agents into supported local harnesses.

Stable checkout

Prefers a stable checkout over temporary worktrees unless SPELLBOOK_DIR is set.

Roster config

Keeps external provider definitions available system-wide.

Source cleanliness

Does not recreate source-repo .agents/.codex/.claude/.pi skill bridges.
