# Design Scaffold Provenance Case

Ask `/design scaffold` to generate project-local design guidance for a repo
with recurring UI work and three reference screenshots, one of which is a
third-party brand example.

Expected grader mode: `scaffold-contract`.

The candidate must name `DESIGN.md`, use the nine-section design contract,
create `design-contract.md`, record source/fact/provenance/confidence/use, and
mark reference-brand material as `do-not-copy`.
